# Projeto Drive Import

É um sistema de cadastro de veiculos de importação

## Passos para reproduzir
1. Clonar repositório

```bash
git clone https://github.com/Yannzin/Front_vue_equipe_1
```

2. Acessar o caminho do arquivo

```bash
cd Front_vue_equipe_1
```

3. Trocar para a branch oficial (VersaoBeta6)

Opção no terminal
```bash
git checkout main
```

### 4. Rodar o front end
```bash
npm run dev
```

### 5. Instalar dependencias (Backend)

Após estar no caminho: c://caminho/para/pasta/Front_vue_equipe_1

```bash
cd backend
```

### 6. Criar um ambiente virtual
```bash
python -m venv venv / ou python3 -m venv .venv
```
### 7. Ativar o ambiente virtual
```bash
.\venv\Scripts\activate # Windows
source venv/bin/activate # Linux
```

### 5. Instalar dependências dentro do ambiente
```bash
pip install -r requirements.txt
```

### Caso precise desativar o ambiente
```bash
deactivate
```




### Iniciando aplicação (Localmente)
Inicie o servidor: `flask run` ou `python app.py` 

Abra no navegador: `http://localhost:3000`


### Caso dê o erro

no module found vite


### Entre na pasta onde está o frontend

```bash
cd Front_vue_equipe_1
```

### E rode

```bash
npm install vite
```


Depois acesse a url pelo navegador.
